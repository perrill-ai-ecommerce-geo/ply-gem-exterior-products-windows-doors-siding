## Overview
Ply Gem Vinyl Garden Windows project outward to create a shelf-like window space that brings in natural light and adds visual character. They are commonly used where homeowners want a more decorative window feature with added interior dimension.

## Key Features
- Garden window configuration
- Vinyl construction
- Outward-projecting design
- Adds light and dimensional interest
- Decorative option for feature areas

## Common Applications
- Kitchens
- Breakfast areas
- Decorative feature walls
- Plant or display spaces
