## Overview
Ply Gem Single Hung Windows feature a classic window style with a fixed top sash and an operable bottom sash. They are a practical choice for homeowners and builders looking for a traditional appearance, dependable ventilation, and compatibility with multiple Ply Gem window series.

## Key Features
- Traditional single hung design
- Operable bottom sash with fixed upper sash
- Available across multiple Ply Gem window series
- Offered in several window material categories depending on series
- Suitable for both new construction and replacement applications

## Common Applications
- Bedrooms
- Living rooms
- Traditional home styles
- New construction projects
- Replacement window upgrades
